Neste laboratório será simulado uma situação de conflito no gitHub e como resolvê-lo.

Programas utilizados: GitHub, Kazam e Meld

1. Foi criado um repositório público, chamado Lab1-SI1 com URL https://github.com/MaianaBt/Lab1-SI1.git

3. Adicionado o arquivo README.txt
4. Adicionado o arquivo Conflito.txt, inicialmente vazio. Esse arquivo será utilizado para simular o conflito.
5. Criada a branch addMaiana

Após o início do vídeo:

6. Foi trocada para a branch addMaiana
7. O arquivo Conflito.txt foi modificado. Foi feito o commit do mesmo.
8. Posteriormente a troca de branch para master.
9. Repete o 7;
10. Foi simulado o conflito ao executar o comando "$ git merge addMaiana"
11. Resolvido localmente utilizando o gedit.
12. Feito novamente o add e commit.
13. Finalmente feito o push e verificado que o arquivo foi modificado no servidor.
